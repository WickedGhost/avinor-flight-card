## Avinor Flight Card

A beautiful and responsive Lovelace card for displaying flight information from the Avinor Flight Data integration.

### Features
- Clean table layout showing flight details
- Real-time updates from Avinor sensors
- Responsive design for all devices
- Easy configuration through Lovelace UI

### Requirements
- Home Assistant 0.115.0+
- Avinor Flight Data integration installed

### Quick Setup
1. Install via HACS
2. Add to your dashboard with entity configuration
3. Enjoy real-time flight information!

For detailed configuration options, see the [README](https://github.com/WickedGhost/avinor-flight-card#readme).
